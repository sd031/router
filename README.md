# Hybrid Router

Cross-platform, extensible router with state and url based routing and built in history

`meteor add hybrid:router`

Platforms Tested
* [x] Web
* [x] iOS
* [x] Android
* [x] Mac OS X
* [x] Chrome Extensions
* [x] Samsung TV
